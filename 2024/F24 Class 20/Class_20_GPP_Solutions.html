Today we will investigate the NumPy module. NumPy is short for Numerical Python, and it is based on arrays.

Start by executing the following, to import numpy as np and to import the array function.

# 1
import numpy as np
from numpy import array

Create a one-dimensional array variable using the array function, in which 5 integers are stored. Also create a 1D array variable that stores 6 floats. Print both arrays and their types.

# 2
int_array = array([11, 2, 33, 9, 4])
float_array = array([1.1, 2.2, 3.3, 4.8, 5.4, 6.2])
print(int_array, type(int_array))
print(float_array, type(float_array))

[11  2 33  9  4] <class 'numpy.ndarray'>
[1.1 2.2 3.3 4.8 5.4 6.2] <class 'numpy.ndarray'>

Use the dtype method to determine the type of both array variables. Note that no arguments are passed to dtype, and the parentheses are not needed.

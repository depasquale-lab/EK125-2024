  2. Check if the key 'pages' exists in the dictionary and print the result.

  3. Use the len() function to determine the number of key-value pairs in the dictionary and print it.

You are managing data about students and their test scores. The data is stored in a dictionary where each key is a student’s name, and the value is another dictionary containing the student’s scores for Math, Science, and English.

Here is the data:

    'Dana': {'Math': 89, 'Science': 94, 'English': 87}
}

  1. Print the scores for 'Alice'.

  2. Retrieve and print the Science score for 'Charlie'.

  3. Add a new student, 'Eve', with the following scores:

  • Math: 88

  • Science: 86

  • English: 89

  1. Calculate and print the average Math score for all students.

  2. Use a loop to print each student’s name and their average score in the format: "Student: <name>, Average Score: <average>".

    'Alice': {'Math': 85, 'Science': 90, 'English': 88},
    'Bob': {'Math': 78, 'Science': 82, 'English': 84},

    'Dana': {'Math': 89, 'Science': 94, 'English': 87}
}

# Step 2: Print the scores for 'Alice'

math_scores = [scores['Math'] for scores in students_scores.values()]

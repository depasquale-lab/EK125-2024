Write a cumulative product function. The function will receive a list of numbers, and will create a new list which is a list of the running products of the numbers in the parameter list. For example, if the list that is passed to the function is [4, 2, 5], the returned list would be [4, 8, 40].

# 8

def cum_prod(num_list):
    """ This function returns a list of the running products of the numbers in the parameter list"""
    prod_list = []
    prod = 1
    for num in num_list:
        prod = prod * num
        prod_list.append(prod)
    return prod_list


print(cum_prod([4, 2, 5]))
print()
help(cum_prod)

Write an “interpolate” function, which takes two tuples, (x0,y0) and (x1,y1), and an input x_i. It will then return the “interpolated” y value, that is to say the one which would correspond to the x_i as if it were on a line. In other words, if (x0,y0) = (0,0), and if (x1,y1) = (1,0.5), then passing x_i=0.5 would result in returning y_i=0.25. And passing x_i=0.9 would return y_i = 0.45.

It should return an error for any x value outside the range [x0,x1].

#12
def interpolate(X0, X1, x_i):
  """ This function interpolates between two points"""
  # Do some bounds checking
  if x_i < X0[0]:
    print ("Error, ", x_i, " is less than ", X0[0])
    return None
  elif x_i > X1[0]:
    print ("Error, ", x_i, " is greater than ", X1[0])
    return None

  # Calculate the slope
  slope = (X1[1]-X0[1])/(X1[0]-X0[0])
  y_i = X0[1] + slope*(x_i - X0[0])

  return y_i

interpolate((10,1),(20,2), 19)

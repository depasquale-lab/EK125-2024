Problem 1: Write a program that takes the total purchase amount and membership status (e.g., “Gold”, “Silver”, “None”) as input. Use nested if statements to calculate and print the final price after applying the appropriate discount based on the membership status. Assume a Gold membership comes with a 15% discount and and a 20% discount on purchases of 100 dollars or more. A Silver membership has an 5% discount and a 10% discount on purchases of 100 dollars or more. Print out the total amount, the discount, and the final price after discount formatted in dollars and cents ($dd.cc). Comment your program accordingly.


# Get user input for total purchase amount and membership status
total_amount = float(input("Enter the total purchase amount: "))
membership_status = input("Enter membership status (Gold/Silver/None): ")

# Initialize discount variable
discount = 0.0

# Determine discount based on membership status
if membership_status == "Gold":
    if total_amount >= 100:
        discount = 0.20  # 20% discount for Gold members on purchases of $100 or more
    else:
        discount = 0.15  # 15% discount for Gold members on purchases below $100
elif membership_status == "Silver":
    if total_amount >= 100:
        discount = 0.10  # 10% discount for Silver members on purchases of $100 or more
    else:
        discount = 0.05  # 5% discount for Silver members on purchases below $100
elif membership_status == "None":
    if total_amount >= 100:
        discount = 0.05  # 5% discount for non-members on purchases of $100 or more
    else:
        discount = 0.0   # No discount for non-members on purchases below $100
else:
    print("Invalid membership status.")

# Calculate final price after discount
final_price = total_amount * (1 - discount)

# Print the final price
print(f"Total amount: ${total_amount:.2f}")
print(f"Discount: {discount * 100:.0f}%")
print(f"Final price after discount: ${final_price:.2f}")

Problem 4: Write a program that solves a quadratic equation of the form ( ax^2 + bx + c = 0 ). The program should prompt the user for the coefficients ( a ), ( b ), and ( c ), and then calculate and print the roots of the equation. Hint: Remember, there are three cases.

import math

# Prompt the user for coefficients a, b, and c
a = float(input("Enter coefficient a: "))
b = float(input("Enter coefficient b: "))
c = float(input("Enter coefficient c: "))

# Calculate the discriminant
discriminant = b**2 - 4*a*c

# Check if the discriminant is positive, zero, or negative
if discriminant > 0:
    root1 = (-b + math.sqrt(discriminant)) / (2*a)
    root2 = (-b - math.sqrt(discriminant)) / (2*a)
    print(f"The roots are real and different: {root1:.2f} and {root2:.2f}")
elif discriminant == 0:
    root = -b / (2*a)
    print(f"The root is real and the same: {root:.2f}")
else:
    real_part = -b / (2*a)
    imaginary_part = math.sqrt(-discriminant) / (2*a)
    print(f"The roots are complex: {real_part:.2f} ± {imaginary_part:.2f}i")

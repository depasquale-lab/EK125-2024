Class 12: Group Practice Problems These will not be handed in, but please take them seriously and do not use generative AI. We have learned some aspects of working with the range function and for loops. Today we will learn more about range, and also the enumerate function, and tuples.

We’ve seen that range(n) generates a sequence of integers 0 through n-1. Specifying two arguments, e.g. range(n,m), generates a sequence of integers n through m-1. The numbers in the sequence get generated one at a time, however. Execute the following:

# 1
range(2,5)

range(2, 5)

We can use the list function to turn this into a list. Execute the following:

#2
list(range(2,5))

[2, 3, 4]

The default step is 1, but by using a third argument a step value can also be specified. Execute the following:

#3
list(range(2,10,2))

[2, 4, 6, 8]

The step value can be negative. For example, range(n,m,-s) generates a sequence of integers from m to n+1 in steps of -s. Execute the following:

# 4
list(range(12,3,-2))


[12, 10, 8, 6, 4]

Now it’s time for you to generate some sequences using list and range. Create the sequence [1, 2, 3, 4, 5].

# 5

[1, 2, 3, 4, 5]

Create the sequence [3, 7, 11, 15, 19].

# 6
list(range(3,20,4))

[3, 7, 11, 15, 19]

Create the sequence [3, 2, 1].

# 7
list(range(3,0,-1))

[3, 2, 1]

What happens when the step goes beyond the specified range? Execute the following:

# 8
list(range(2, 4, 5))

[2]

The enumerate function returns the index and item from a sequence. Execute the following:

# 12
list(enumerate([3, 2, 11]))

[(0, 3), (1, 2), (2, 11)]

This actually created a list of tuples (which we’ll cover shortly!). In each, you have the index and then the item from the list. Let’s use this in a for loop. Execute the following:

# 13
for index, item in enumerate([3, 2, 11]):
    print(index, item)

0 3
1 2
2 11

Create a list of strings. Then, for each string, print the index of the string (naturally, e.g. 1,2,3.. instead of 0,1,2..), the number of characters in the string, and the individual characters. For example, if the first string is “Hello”, the first line of output would be “Word 1 “Hello” has 5 characters: H e l l o”

# 16
strlist = ["Hello", "Hi", "Ciao"]
for i, s in enumerate(strlist):
    print(f'Word {i+1} "{s}" has {len(s)} characters:', end=' ')

        print(c, end=' ')
    print()

Word 2 "Hi" has 2 characters: H i 
Word 3 "Ciao" has 4 characters: C i a o

# 4

b = [2, 11, -5, 4, 33]

Problem 5: It is frequently useful to calculate the differences between consecutive elements in a list of numbers. For example, if the list is [5, 11, 2, 33, -4], the differences would be [11-5, 2-11, 33-2, -4-33], or [6, -9, 31, -37]. For a list variable that has a length of n, the resulting differences list would have a length of n-1. Using a list variable numlist, start by creating two list variables: one stores all but the first element, and the other stores all but the last element. Then, using the method above, use a for loop and enumerate to print the differences.

# 5
numlist = [5, 11, 2, 33, -4]
list1 = numlist[1:]
list2 = numlist[:len(numlist)-1]
for i, item in enumerate(list1):
    print(item - list2[i], end = ' ')
print()

6 -9 31 -37

Problem 7: Create a nested tuple of numbers. Find the overall maximum.

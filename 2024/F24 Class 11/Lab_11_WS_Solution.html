If you want to use a built in function, you can simply use the function name as the key (i.e. key=sum) You can also define a custom function with the syntax: key=lambda x: + your definition here (i.e key=lambda x: x.lower()) Try defining some of your own keys to sort lists by different criteria. For example, sort a list of strings by length, or by last character.

# sorting by last character
new_string=sorted(strings, key=lambda x: x[-1])

Problem 6: Write a Python program that defines a list of numbers. The program should prompt the user to enter a number and a position, then insert the number at the specified position in the list. Finally, print the updated list.

# 6
# Define the list of numbers
numbers = [10, 20, 30, 40, 50]

# Prompt the user to enter a number and a position
num = int(input("Enter a number to insert: "))
position = int(input("Enter the position to insert the number at: "))

# Check if the position is valid
if 0 <= position <= len(numbers):
    numbers.insert(position, num)
    print("Updated list:", numbers)
else:
    print("Invalid position.")

Problem 7: Write a Python program that defines a list of numbers. The program should prompt the user to enter a position and remove the element at that position using the pop method. If the position is invalid, display an appropriate message. Finally, print the updated list.

# 7
# Define the list of numbers
numbers = [11, 22, 33, 44, 55]

# Prompt the user to enter a position
position = int(input("Enter the position to remove an element from: "))

# Check if the position is valid and pop the element
if 0 <= position < len(numbers):
    removed_element = numbers.pop(position)
    print(f"Removed element: {removed_element}")
    print("Updated list:", numbers)
else:
    print("Invalid position.")

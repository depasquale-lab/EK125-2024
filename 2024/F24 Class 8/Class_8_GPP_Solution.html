You’re an overworked and underappreciated server at The Green Midget Cafe. Trying to repeat the menu to customers over a bunch of chanting Vikings is straining your voice, so you want to automate generating the menu.

  9. Spam, spam, spam, spam, spam, spam, spam, baked beans, spam, spam, spam, and spam

  10. Spam, sausage, spam, spam, spam, bacon, spam, tomato, and spam

Write a script that will generate each menu item from the list provided and print them as a numbered list so your customers can simply order by number. Also, you’re British, so that Oxford comma is important!

#10
# Use this list of items to create your nested loops
ingredients = ['eggs', 'bacon', 'sausage', 'spam', 'baked beans', 'tomato']
indices = [[0,1], [0,2,1], [0,3], [0,1,3], [0,1,2,3], [3,1,2,3], [3,0,3,3,1,3], [3,3,3,0,3], [3,3,3,3,3,3,3,4,3,3,3,3], [3,2,3,3,3,1,3,5,3]]
menu = [''] * len(indices)  # Initialize menu with empty strings

for i in range(len(indices)):
    if len(indices[i]) == 2:
        menu[i] = f'{ingredients[indices[i][0]]} and {ingredients[indices[i][1]]}'
    else:
        for j in range(len(indices[i]) - 1):
            menu[i] += ingredients[indices[i][j]] + ', '
        menu[i] += 'and ' + ingredients[indices[i][-1]]
    print(f"{i + 1}. {menu[i]}")

print(f"Total menu items: {len(menu)}")
